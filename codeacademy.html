Estados Unidos, México, Colombia, Ir al Sitio Web de Code Academy

Code Academy es uno de los sitios más populares donde las personas aprenden a programar de forma gratuita. Más de 24 millones de personas han aprendido a programar utilizando esta plataforma.

Ofrecen cursos que van desde la programación introductoria hasta lenguajes más avanzados. El enfoque de aprendizaje interactivo te permite aplicar de inmediato lo que has aprendido. Hay más de 300 millones de horas de contenido de programación gratis disponibles para que puedas pasar de ser un principiante absoluto a un programador experimentado.

Algunos de los lenguajes que podrás aprender en Code Academy incluyen: HTML JavaScript CSS jQuery PHP Python Ruby También hay muchos otros cursos disponibles, pero los lenguajes anteriores son el enfoque principal.
